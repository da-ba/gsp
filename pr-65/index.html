Available Commands

/

Show the command palette with all available commands

/giphy [search]

Search and insert GIFs from Giphy

/emoji [search]

Search and insert emojis

/font [style]

Apply text styles, colors, and sizes

/mermaid [type]

Insert Mermaid diagram templates

/now [format]

Insert formatted timestamps

/kbd [shortcut]

Format keyboard shortcuts

/link [url]

Insert markdown links with titles

/mention [user]

Mention GitHub users

Note: This is a live demo of the GitHub Slash Palette extension. The full extension works on GitHub.com and includes additional features like CI link integration.
